Приветствую! Итак чтобы начать использовать бота 
1.Вам нужен язык прогромирования Python рекомендую по новее но так как на более старые пк новые не установятся вообще. Поэтому устанавливаем версию по ссылке https://www.python.org/downloads/release/python-380/ как сразу становили ОБЯЗАТЕЛЬНО СТАВИТЕ 2 ГАЛОЧКУ НА Python To PATH или как там!!!!! Это обязательно. Для проверки в cmd пишите python и если написана версия то всё сделано правильно
2.Нужно установить библиотеки. Чтобы установить нужные библиотеки из файла dowloand.bat
3.Открываете любой текстовый редактор для кода или обычный блокнот именно через кнопку "Изменить" через правую кнопку мыши
4.Создаёте токен на сайте https://discord.com/developers/applications
5.Запускаем bot.bat или bot.py для работы бота
Спасибо за прочитание посмотреть подробно: https://youtu.be/o6eOYhIHl4o
Если будут ошибки пишите в коментах попробую помочь
Хочю предупредить про библиотеку aiohttp она используется для того чтобы бот отправлял сообщения с картинкой но она у некоторых может работать
